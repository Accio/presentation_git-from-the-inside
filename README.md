GIT from the inside
============

This presentatation is used in  my _git from the inside_ talks. 

HOW TO USE IT
---------------
 * Open `output/index.html` in a modern browser.
OR
 * [neuhalje.github.io/git-from-the-inside](http://neuhalje.github.io/git-from-the-inside)

Building
==========

```sh
grunt
```

Install Build ENV
---------------------
```sh
# @pre: need npm
npm install
grunt
```
